# Dify Landing Page

This project is a simple landing page that integrates an AI assistant chat feature using an iframe. The page is designed with a clean interface to provide a user-friendly experience.

## Project Structure

```
dify-landing
├── src
│   ├── index.html          # Main HTML document for the landing page
│   ├── components
│   │   └── chat-embed.html # Contains the iframe for the AI assistant chat
│   ├── css
│   │   └── styles.css      # Styles for the landing page
│   └── js
│       └── main.js         # JavaScript for interactivity
├── .gitignore              # Files and directories to ignore by Git
├── package.json            # Configuration file for npm
└── README.md               # Documentation for the project
```

## Getting Started

To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd dify-landing
   ```

2. **Install dependencies**:
   If you have npm installed, run:
   ```bash
   npm install
   ```

3. **Open the landing page**:
   Open `src/index.html` in your web browser to view the landing page.

- Clean and simple interface
